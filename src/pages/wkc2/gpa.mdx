# Зона Компримирование (ГПА) WKC-2

<GpaAddwkc2 />

<HeroPattern />
Газоперекачивающий агрегат (ГПА) - это устройство, которое используется для перекачки
газа по трубопроводу. Он обычно состоит из компрессора и приводного механизма, такого
как электродвигатель или газовая турбина. ГПА обеспечивает необходимое давление и
поток газа для перемещения его из одной точки в другую.

<div className="my-16 xl:max-w-none">
  <Compressor1 />
</div>

В состав ГПА входит газотурбинная установка

- PGT25+SAC с 17-ступенчатым компрессором, со степенью сжатия 23:1, Скорость вращения газовой турбины 10000 об/мин, скорость вращения силовой турбины 6100 об/мин, тепловая эффективность 34,1%, Объем расходуемого газа 117 Нм3/сут., КПД компрессора 88%.
